# أكاديمية الفهم — تعلّم التداول بمنهجية ICT / Smart Money Concepts

منصة ويب متكاملة (Next.js) لتعليم التداول من الصفر إلى الاحتراف باستخدام
منهجية ICT وSmart Money Concepts، مدعومة بمرشد ذكاء اصطناعي عبر OpenRouter،
وتدريب عملي على شارت حي ببيانات حقيقية.

## المحتوى

- **منهج كامل**: 6 مستويات × 3 دروس (18 درساً) من الأساسيات إلى الاحتراف،
  كل درس يتضمن شرحاً، نقاطاً أساسية، واختباراً قصيراً.
- **تتبع تقدم**: يُحفظ تلقائياً في متصفحك (localStorage)، بدون الحاجة لحساب أو قاعدة بيانات.
- **مرشد ذكي**: صفحة `/mentor` للمحادثة الحرة، وصفحة `/practice` لمراجعة تحليلك على شارت حقيقي.
- **شارت تدريبي**: بيانات شموع حقيقية من Binance (BTC/USDT وETH/USDT) عبر `lightweight-charts`.

## التشغيل محلياً

المتطلبات: Node.js 18 أو أحدث.

```bash
npm install
cp .env.example .env.local
```

افتح `.env.local` وضع مفتاح OpenRouter الخاص بك:

```
OPENROUTER_API_KEY=sk-or-v1-...
```

يمكنك الحصول على مفتاح من [openrouter.ai/keys](https://openrouter.ai/keys).

ثم شغّل:

```bash
npm run dev
```

افتح المتصفح على `http://localhost:3000`.

## النشر (اختياري)

المشروع مبني بـ Next.js 14 (App Router) ويعمل مباشرة على منصات مثل Vercel:

1. ارفع المشروع إلى مستودع GitHub.
2. اربطه بحساب Vercel واستورده كمشروع جديد.
3. أضف متغيرات البيئة نفسها (`OPENROUTER_API_KEY`, `OPENROUTER_MODEL`, `SITE_URL`) من إعدادات المشروع في Vercel.
4. انشر — لا حاجة لأي إعداد إضافي.

**مهم:** مفتاح OpenRouter يُستخدم فقط من جانب الخادم (`app/api/chat/route.js`)
ولا يظهر أبداً في كود الواجهة الأمامية، حفاظاً على أمانه.

## تخصيص المحتوى

- **إضافة/تعديل الدروس**: كل شيء في `lib/curriculum.js` — مصفوفة واحدة منظمة بمستويات ودروس، سهلة التوسعة.
- **تغيير شخصية المرشد الذكي**: عدّل `SYSTEM_PROMPT` في `app/api/chat/route.js`.
- **تغيير النموذج المستخدم**: غيّر `OPENROUTER_MODEL` في `.env.local` إلى أي معرّف نموذج من [openrouter.ai/models](https://openrouter.ai/models).
- **الهوية البصرية**: الألوان والخطوط معرّفة في `tailwind.config.js` و`app/globals.css`.

## بنية المشروع

```
app/
  page.js                 الصفحة الرئيسية
  curriculum/              المنهج (قائمة المستويات → دروس كل مستوى → صفحة الدرس)
  practice/page.js         التدريب العملي (شارت + مراجعة AI)
  mentor/page.js           المحادثة الحرة مع المرشد
  api/chat/route.js        وسيط آمن مع OpenRouter
lib/
  curriculum.js            محتوى المنهج الكامل
  progress.js               تتبع التقدم عبر localStorage
components/                جميع مكونات الواجهة القابلة لإعادة الاستخدام
```
